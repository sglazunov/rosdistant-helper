# Дисклеймер и условия использования

**Кратко:** инструмент учебный, для личного использования. Он не взламывает
платформу и не обходит авторизацию — он работает только через вашу собственную
залогиненную сессию. Ответственность за использование — на вас.

## Что это и чего это не делает

«Помощь росдистантикам / Rosdistant Helper» помогает студенту **сохранить для
личного офлайн-доступа** учебные материалы платформы Росдистант, к которым у него
**уже есть легальный доступ** после входа в свой аккаунт.

Расширение:

- **не** взламывает платформу, не подбирает и не крадёт пароли;
- **не** обходит авторизацию — оно использует cookie вашей **текущей** сессии,
  ровно как это делает сам браузер, когда показывает вам учебник;
- **не** отправляет ваши данные, файлы или пароли куда-либо — вся работа идёт
  локально в браузере;
- работает с теми материалами, которые платформа уже отдала вашему браузеру.

«Пароль», который расширение показывает/использует, — это пароль самого
документа, который платформа **сама сохраняет в вашем браузере** (`localStorage`)
для отображения учебника. Расширение его не подбирает.

## Ваша ответственность

- Используйте инструмент **только** для материалов, к которым у вас есть
  правомерный доступ.
- Соблюдайте **авторские права** правообладателей и **условия использования**
  платформы Росдистант. Загруженные материалы могут быть защищены авторским
  правом — не распространяйте их незаконно.
- Не используйте инструмент для массового скачивания, перепродажи или публичного
  распространения чужих материалов.
