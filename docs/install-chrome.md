# Установка в Chrome / Yandex / Edge / Opera

Все эти браузеры основаны на Chromium и используют **одну и ту же папку**
`dist/chrome`.

## Шаги

1. Скачайте этот репозиторий (кнопка **Code → Download ZIP** на GitHub) и
   распакуйте его. Либо `git clone`.
2. Откройте страницу расширений:
   - **Chrome** — `chrome://extensions`
   - **Yandex** — `browser://extensions`
   - **Edge** — `edge://extensions`
   - **Opera** — `opera://extensions`
3. Включите **Режим разработчика** (переключатель сверху справа).
4. Нажмите **Загрузить распакованное расширение** и выберите папку
   **`dist/chrome`** из этого проекта.
5. Значок расширения появится на панели. Закрепите его (иконка пазла → булавка).

## Как пользоваться

1. Войдите в Росдистант (`https://edu.rosdistant.ru`) как обычно.
2. **Учебник по ссылке:** нажмите на значок расширения, вставьте ссылку на
   учебник и нажмите **«Скачать учебник»**.
3. **Учебник с текущей страницы:** откройте сам учебник, нажмите на значок и
   **«Скачать учебник»** с пустым полем ссылки.
4. **Тесты:** откройте завершённую попытку теста, раскройте «Экспорт тестов» и
   выберите HTML или PDF.

## Обновление

После `git pull` или новой сборки откройте страницу расширений и нажмите
кнопку **обновления** (↻) на карточке расширения.
